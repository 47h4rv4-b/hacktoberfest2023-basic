# Divide a by b without using divide , multiplication or modulus operator.

## A brief overview

Given two integers a and b, divide a by b without using divide , multiplication or modulus operator (unorthodox way).

## Sample test cases

```
input: 21 3
output: 7 0
```

```
input: 55 6
output: 9 1
```

```
input: 165 19
output: 8 13
```
